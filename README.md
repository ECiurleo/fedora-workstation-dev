# fedora-workstation-dev
Setup for a fedora workstation to install and configure a developer machine

## Installs
Browser:
* Microsoft Edge

Dev Tools:
* Docker Desktop
* JetBrains Toolbox
* Microsoft VSCode
* GitHub Desktop

ClI Tools:
* Azure CLI
* AWS CLI
* Kubectl
* Helm
* K9s

Chat Apps:
* Slack
* Zoom
* Teams
* Telegram

Other
* Gnome Tweaks/Extensions
* Yubikey Authenticator
* Logitech controller
* Steam
